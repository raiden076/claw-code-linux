🦀 Claw

Claude Code CLI - AI-powered coding assistant for your terminal

Quick Install

curl -sSL https://claw.arkaprav0.in/claw-install.sh | bash

Manual Install

Download the latest release for your architecture:

# x86_64 (AMD64)
wget https://github.com/raiden076/claw-code-linux/releases/latest/download/claw-x86_64-unknown-linux-gnu.tar.gz

# aarch64 (ARM64)
wget https://github.com/raiden076/claw-code-linux/releases/latest/download/claw-aarch64-unknown-linux-gnu.tar.gz

Usage

# Interactive REPL
claw

# Single prompt
claw "explain this codebase"

# With specific model
claw --model claude-opus "refactor main.rs"
